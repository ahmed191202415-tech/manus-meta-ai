# تعريفات المقاييس

- ctr_link = link_clicks / impressions: قياس جاذبية الإعلان للنقر.
- outbound_ctr = outbound_clicks / impressions: قياس نية الخروج من ميتا.
- lpv_rate = landing_page_views / outbound_clicks: كشف التسريب بعد الضغط.
- atc_rate = add_to_cart / landing_page_views: قوة العرض أو صفحة المنتج.
- checkout_rate = initiate_checkout / add_to_cart: احتكاك ما قبل الدفع.
- purchase_rate = purchases / landing_page_views: تحويل الزيارات الفعلية.
- cpa = spend / results: تكلفة النتيجة حسب نوع الحملة.
- roas = purchase_value / spend: العائد عندما تتوفر قيمة.
- signal_quality = meaningful_results / outbound_clicks: جودة الإشارة بعد النقر.
