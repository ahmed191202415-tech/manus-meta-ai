# العلاقات بين الأرقام

- frequency يزيد + ctr يقل + cpa/cpm يزيد = إجهاد محتوى أو تشبع جمهور.
- outbound_ctr جيد + lpv_rate ضعيف = تسريب بعد الضغط أو مشكلة تتبع/سرعة/وجهة.
- ctr جيد + cvr/result_rate ضعيف = فضول بلا نية أو عدم تطابق الوعد.
- spend يزيد + results لا تزيد = هدر أو عائد حدي ضعيف.
- cpm يزيد + ctr يقل = ضغط مزاد أو جودة إعلان ضعيفة.
- roas يقل مع spend يزيد = توسع غير صحي أو جمهور محدود.
