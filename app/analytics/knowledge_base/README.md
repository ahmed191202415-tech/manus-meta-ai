# Meta Ads Intelligence Knowledge Base

قاعدة معرفة مختصرة ومنظمة لتغذية طبقات التحليل والتقرير.

## الملفات

1. `01_principles.md` — فلسفة التحليل وقاعدة عدم الهبد.
2. `02_metrics.md` — تعريفات المقاييس ومعانيها.
3. `03_relationships.md` — العلاقات الثنائية والثلاثية.
4. `04_creative_content.md` — تشخيص المحتوى والكرياتيف.
5. `05_campaign_types.md` — ما يظهر وما يمنع حسب نوع الحملة.
6. `06_tracking_budget_reporting.md` — القياس، الميزانية، والتقرير.

## قاعدة الاستخدام

- الكود يحسب ويستخرج الأدلة.
- قواعد التشخيص تحدد السيناريوهات.
- هذه المعرفة تشرح المعنى والقرار بالعربي.
- لا تستخدم المعرفة لعرض قسم بلا بيانات.
