# GA4 Metrics Dictionary

- `sessions`: visits started on the site or app.
- `activeUsers`: users who had an engaged session or meaningful activity.
- `engagedSessions`: sessions that met GA4 engagement criteria.
- `engagementRate`: engaged sessions divided by sessions.
- `conversions`: configured conversion events in GA4.
- `totalRevenue`: revenue reported by GA4.
- `averageSessionDuration`: average session duration.
- `eventCount`: number of events.

Use backend-calculated metrics as the source of truth. Do not recalculate totals in the GPT response unless the backend returns raw rows only.
