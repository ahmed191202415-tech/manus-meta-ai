# Website Analysis Playbook

Start with tracking quality, then traffic volume, engagement, conversion rate, and device/source splits.

Do not make strong performance decisions when tracking quality is weak or sample size is low.

Common decisions:
- Fix Tracking: conversion or lead events are missing.
- Fix Landing Page: traffic exists but engagement or conversion is weak.
- Fix Mobile UX: mobile conversion rate is meaningfully below desktop.
- Hold: data is inconclusive.
- Scale: only when tracking is reliable and conversion economics are strong.
